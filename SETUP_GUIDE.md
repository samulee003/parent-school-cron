# 企業微信群機器人設定指南（小白版）

> 10 分鐘搞定，之後完全不用管

---

## 第一步：註冊企業微信（5 分鐘）

1. 打開瀏覽器，進入 https://work.weixin.qq.com/
2. 點「立即註冊」
3. 選「團隊」→ 填你的名字和手機號
4. 不需要公司營業執照，**個人就能註冊**
5. 註冊完會收到企業微信 App 下載連結（手機也要裝）

> 💡 如果你已經有企業微信帳號，跳過這步

---

## 第二步：建群 + 加機器人（3 分鐘）

### 2.1 建群

在手機企業微信 App 裡：

1. 點右上角「+」→「发起群聊」
2. 隨便拉一個人進群（之後可以移除），或者選「仅自己」
3. 群名建議取：**家長學堂課程推送**

### 2.2 添加群機器人

1. 進入群 → 點右上角「...」
2. 找到「群機器人」→ 點進去
3. 點「添加機器人」
4. 機器人名字填：**家長學堂助手**
5. **複製 Webhook URL**（長這樣）：
   ```
   https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
   ```

> ⚠️ 這個 URL 很重要，待會要貼到 Zeabur，**不要外洩**

### 2.3 關閉成員發言（讓群安靜）

1. 進入群 → 點右上角「...」
2. 找到「群管理」→ 開啟「僅管理員可發言」
3. 這樣家長進群後只能看，不能聊

---

## 第三步：生成邀請二維碼（1 分鐘）

1. 進入群 → 點右上角「...」
2. 找到「群二維碼」→ 保存圖片
3. 這個二維碼就是給家長掃的

> 家長用**普通微信**掃碼就能進群，不需要裝企業微信

---

## 第四步：把 Webhook URL 告訴我

把你複製的 Webhook URL 發給我，我來設置 Zeabur 環境變量。

格式是這樣：
```
https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxxxxx
```

## 常見問題

**Q: 家長怎麼進群？**
A: 你把群二維碼發給他們，普通微信掃碼就能進。

**Q: 家長能在群裡說話嗎？**
A: 如果你開了「僅管理員可發言」，就不能。推薦開。

**Q: 我需要每天維護嗎？**
A: 不需要。設好之後全自動，除非 DSEDJ 網站改版。

**Q: Webhook URL 洩漏了怎麼辦？**
A: 在群機器人設定裡重新生成一個，然後更新 Zeabur 環境變量。

**Q: 推送時間能改嗎？**
A: 能，改 Zeabur 的 `WXAGENT_PUSH_HOUR` 環境變量就行。
